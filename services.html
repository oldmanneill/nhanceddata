Service Product Suite
Pick and choose from any of the following to create a product suite that fits your needs.
Service Description
Support Assistance Provide assistance with working errors (especially helpful after conversions, go-lives, upgrades)
Support Training Training on maintaining interfaces, working errors
Documentation Provide detailed documentation on interfaces used including fields/formats and customizations as well as workflows/test case scenarios.
Interface documentation is especially helpful for reference when upgrading, converting, or switching vendors. Documentation can be used to compare differences in workflow outcomes.
Conversion/Migration Assistance Assistance with converting data with EHRs, changing interface engines, reinstallations/recovery.
For Touchworks EHR users, it is recommended to purchase the client-led Rhapsody package from Allscripts.
Interface Implementation Assistance Provide knowledge transfer and implementation strategies for clients creating their own interfaces for Touchworks EHR
Clients will need to have purchased the interface education package from Allscripts. This assistance can be especially beneficial if the previously trained interface specialist has left your department as the education license stays with the department, not the individual trained.
Interface Optimization/Customization Improve inefficient interfaces that may be slow or error out frequently or make use of new fields and function
